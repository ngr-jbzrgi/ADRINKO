## Installation

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm run dev
```

3. Open your browser and visit `http://localhost:5173`

## Build for Production

```bash
npm run build
```

The production-ready files will be in the `dist` folder.

## Preview Production Build

```bash
npm run preview
```

## Customization

### Colors
Edit `tailwind.config.js` to customize the ice and frost color palettes.

### Animations
Modify animation durations and effects in individual components using Framer Motion props.

### Content
Update product information, company details, and contact info in the respective component files.
